Hier kommen zukünftig die einzelnen 3D-Modelle rein! ;-)

Gängige Modellformate sind:
* GLTF (entweder als einzelne Datei mit Endung GLB oder als Gruppe mehrerer Dateien, in der eine Datei die Endung GLTF hat)
* OBJ (Wavefront object) i.V.m. MTL-Dateien und externen Texturen
* FBX (Filmbox) i.V.m. externen Texturen
* DAE (Collada) i.V.m. externen Texturen
